# MS
A small microservice architecture based application for students
This application will contain multiple services:

**1 . Students :**

    -> Add a student
    -> Update a student
    -> Delete a student
    -> Get a student
    -> Add subject to a student
    -> Remove subject from a student
    -> Get subjects for a student
    
**2. Subjects  :**

    -> Add a subject
    -> Update a subject
    -> Delete a subject
    -> Get a subject
